# Rock-Paper-Scissors-Game
A classic Rock Paper Scissors game built using HTML, CSS, and JavaScript. Challenge the computer, track your score, and enjoy a simple and responsive user interface.

## Features
- Interactive and engaging gameplay
- Real-time score tracking
- Random computer choices
- Responsive design for different screen sizes
- Modern and user-friendly interface

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
